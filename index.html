About This Tool

            The Recon Tool is a web-based application designed for ethical reconnaissance purposes. It provides users with essential website information such as Whois details and subdomain enumeration. This tool is developed with simplicity and efficiency in mind, aiming to assist cybersecurity enthusiasts and professionals in gathering publicly available data for authorized purposes.
            Purpose  
            The Recon Tool is built to:  
            - Help cybersecurity professionals perform preliminary reconnaissance for security assessments.  
            - Provide domain-related details, including registrar information, creation dates, and expiration dates.  
            - Demonstrate how Python can power backend logic for reconnaissance in a secure and ethical manner.  
            - Serve as a learning project to understand web development and cybersecurity integration.  
            
            
Key Features
  
            
1. Whois Lookup:
  
               - Extracts information about the domain name, registrar, and registration dates.  
               - Useful for understanding the ownership and lifecycle of a domain.  
            
            2. Subdomain Enumeration:  
               - Returns a list of subdomains (currently dummy data for demonstration purposes).  
               - Can be extended with actual subdomain enumeration logic using tools like dnspython or external APIs.
